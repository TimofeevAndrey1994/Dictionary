# Dictionary

Dictionary — Android-приложение словаря, которое находится в процессе разработки.

Проект создаётся как pet-project для практики современной Android-разработки.  
Сейчас в приложении уже подготовлены базовая структура проекта, сетевой слой и доменная модель для работы со словарным API.  
Дальнейшая разработка планируется с использованием **Jetpack Compose**.

---

## Текущий статус проекта

Проект находится в активной разработке.

На данный момент реализовано:

- базовая архитектурная структура приложения
- интеграция с внешним словарным API
- сетевой слой на **Retrofit**
- модели ответа API (**DTO**)
- маппинг сетевых моделей в доменные модели
- `Repository` и `UseCase` для получения данных о слове
- обработка результата запроса через `Resource`

Сейчас основная цель — продолжить разработку интерфейса и перевести проект на **Jetpack Compose**.

---

## Планируемый функционал

В приложении планируется реализовать:

- поиск слова
- отображение транскрипции
- вывод значений и определений
- отображение примеров использования
- работа с частями речи
- история поиска
- избранные слова
- современный UI на **Jetpack Compose**

---

## Архитектура

На текущем этапе проект уже разделён на основные слои:

- `data` — работа с API, DTO, мапперы, реализация репозитория
- `domain` — модели, интерфейсы репозитория и use case
- `di` — создание зависимостей
- `utils` — вспомогательные классы для обработки результата

Такое разделение позволяет дальше развивать проект и постепенно переносить интерфейс на **Jetpack Compose**.
